# Lab Report - C/C++ Assignments

This repository contains C/C++ lab assignments from second semester coursework.

## Files

- **lab.cpp** - Main lab assignment
- **lab2.cpp** & **lab2.c** - Lab 2 exercises (C and C++ versions)
- **quad.cpp** - Quadratic equation solver
- **GG.cpp** - Additional programming exercise
- **corssprocct.c** - Cross-product computation
- **whta.c** - Miscellaneous C program
- **output/** - Compiled executable files

## Compilation

To compile the programs:

```bash
# C programs
gcc filename.c -o output/filename.exe

# C++ programs
g++ filename.cpp -o output/filename.exe
```

## Execution

```bash
./output/filename.exe
```
